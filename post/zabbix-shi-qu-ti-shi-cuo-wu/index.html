    • zabbix时区提示错误

    更改php.ini配置文件的设置，去掉date.timezone的注释，将时区改为Asia/Shanghai

    vim /etc/php.ini 

    [Date]

; Defines the default timezone used by the datefunctions

; http://php.net/date.timezone


date.timezone=Asia/Shanghai

    将最前面的分号去掉，上面时区改成亚洲上海。

    查看并修改/etc/httpd/conf.d/zabbix.conf配置文件的配置

    vim /etc/httpd/conf.d/zabbix.conf 

    Options FollowSymLinksAllowOverride None

Order allow,deny

Allow from all

php_value max_execution_time 300

php_value memory_limit 128M

php_value post_max_size 16M

php_value upload_max_filesize 2M

php_value max_input_time 300

php_value date.timezone Asia/Shanghai

    重启httpd服务

    systemctl restart httpd

    重新刷新网页即可
